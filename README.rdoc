== Open KVB API Project

This is an inofficial KVB API Project. Only a prototype. Use it as it is.

== Usage

=== Request

  http://kvb-api.herokuapp.com/stations/:ID(.json)

  e.g.: http://kvb-api.herokuapp.com/stations/236.json
